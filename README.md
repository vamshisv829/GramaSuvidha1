# GramaSuvidha1 🌾

GramaSuvidha1 is an Android application designed to improve transparency and digital access for rural infrastructure and Panchayat development projects. The app helps citizens track village-level works, monitor progress, and stay informed about community development activities.

---

## 📱 Features

- 🏗️ Track village infrastructure projects
- 📍 View project status and progress updates
- 📸 Upload and monitor work images
- 🔔 Real-time notifications and updates
- 👥 Citizen engagement and transparency
- ☁️ Firebase backend integration
- 📊 User-friendly Android interface

---

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Android Studio
- JDK 17+
- Firebase Account
- Git

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/vamshisv829/GramaSuvidha1.git
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Connect Firebase:
   - Add your `google-services.json` file inside the `app/` folder

5. Run the application on an emulator or Android device

---

## 🔥 Firebase Setup

Enable the following Firebase services:

- Authentication
- Firestore Database
- Storage
- Realtime Database
